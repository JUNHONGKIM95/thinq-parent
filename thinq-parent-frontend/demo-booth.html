📅
Core Feature
캘린더 & TODO
임신 D-Day와 가족 일정을 한곳에서 관리하고, 주차별 맞춤 할 일을 추천받아 완벽히 준비된 상태로 출산을 맞이합니다.
📆
D-Day & 가족 공유 캘린더
출산 예정일 카운트다운과 아기·가족·업무·개인 카테고리별 컬러로 일정을 가족 모두가 실시간 확인
✅
주차별 To-Do 추천
보건복지부 육아 가이드 기반의 주의사항·정기진찰·기본검사 항목을
해당 주차에 맞게 제공
☑️
체크리스트
출산 준비물 등 가족이 함께 체크하는 공유 리스트
👥
Community
커뮤니티
약 6만 개의 임신·육아 전문 커뮤니티 게시글을 분석하여 Needs/Pain 카테고리를 구성하고, 익명 기반으로 자유롭게 소통하는 공간입니다.
📝
익명 커뮤니티 글쓰기
성향·Needs/Pain 카테고리 기반으로 익명 글을 작성하고 자유롭게 소통
🏷️
MomBTI 태그 필터링
작성자명 옆에 MomBTI 유형 태그가 표시되어 유사 성향 엄마를 빠르게 식별
📊
데이터 기반 카테고리
약 6만 개 임신·육아 전문 커뮤니티 게시글을 분석하여 최적의 카테고리 구성
🏠
Smart Home
가전 제품 루틴
임신·출산 전문 커뮤니티 크롤링으로 수집한 주차별 실제 키워드를 분석해, 임신 초·중·후기별 맞춤 루틴을 LG 가전과 연동하여 제안합니다.
🔍
커뮤니티 키워드 크롤링
임신·출산 전문 커뮤니티에서 주차별 실제 키워드를 수집·분석하여 데이터 기반 루틴 설계
📊
초·중·후기별 맞춤 루틴
임신 시기별 키워드 분석 결과를 바탕으로 세탁·공기질·온도 등 최적의 가전 루틴 자동 추천
⚙️
LG 가전 자동 제어
추천 루틴을 세탁기·건조기·공기청정기 등 LG 가전에 연동하여 원터치로 실행
🧠
Personality
MomBTI 테스트
간단한 테스트로 나만의 육아 성향을 분석하고, 4가지 축 기반의 MomBTI 유형과 8가지 특성을 시각적으로 확인할 수 있습니다.
📋
MomBTI 요약 및 정의 제공
테스트 결과로 사용자의 육아 성향을 요약·정의하고 상세하게 설명
📊
4가지 축 비율 분석
DB 기반으로 4가지 성향 축의 비율을 계산하여 정량적 결과 제공
🔎
8가지 특성 시각화
특성을 상세 설명하고, 사용자에게 해당하는 특성을 시각적으로 강조 표시
📝
Record
임신 다이어리
소중한 임신 기간의 매일을 사진과 글로 기록하고, 앨범으로 남기는 감성적 케어 기능입니다.
✍️
일기 작성·조회
날짜별로 임신 일기를 작성하고, 지난 기록을 언제든 돌아볼 수 있는 타임라인
📷
사진 첨부 기록
초음파 사진, 배 사진 등 소중한 순간을 이미지와 함께 기록
💝
배우자와 일기 공유
배우자와 함께 임신 기간의 추억을 기록하고 공유
💬
AI Assistant
AI 전문가 챗봇
임신·육아 관련 궁금증이 생길 때 필요시 즉시 접근할 수 있는 맞춤 지식 응답 인터페이스입니다.
⚡
즉시 접근 가능
플로팅 버튼으로 어느 화면에서든 바로 챗봇을 호출하여 질문 가능
🎯
증상별·주차별 맞춤 응답
보건복지부 육아가이드와 충청북도 지역장애인보건의료센터 자료를
RAG 기술로 학습해 신뢰할 수 있는 답변 제공
📖
꼼꼼하고 상세하게
간단한 궁금증부터 복잡한 질문까지 자세히 안내
산모에게 민감하거나 위험할 수 있는 사항은 전문 의료진 상담을 권유합니다.
