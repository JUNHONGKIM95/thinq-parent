Core Feature
캘린더 & TODO
임신 D-Day와 가족 일정을 한곳에서 관리하고, 주차별 맞춤 할 일을 추천받아 준비 누락 없이 출산을 맞이합니다.
📆
D-Day & 가족 공유 캘린더
출산 예정일 카운트다운과 아기·가족·업무·개인 카테고리별 컬러 코드 일정을 가족 모두가 실시간 확인
✅
주차별 To-Do 추천
병원 일정, 검사 체크, 영양 관리, 준비물, 주의 사항 등 임신 주차에 맞는 맞춤 할 일 자동 추천
💌
응원 메시지 & 체크리스트
남편의 따뜻한 응원 메시지와 출산 준비물 등 가족이 함께 체크하는 공유 리스트
Key Value
막연한 불안감을 해소하고, 임신·출산을 가족 모두의 여정으로 만들어 동반자적 관계를 형성합니다.
👥
Community
커뮤니티
비슷한 상황과 성향의 엄마들이 모여 경험을 공유하고, 깊은 정서적 공감대를 형성하는 임산부 맞춤 소통 공간입니다.
📝
게시글 작성 & 소통
임신·출산 경험을 자유롭게 나누고, 좋아요·댓글로 서로 응원하며 소통
🏷️
MomBTI 태그 필터링
작성자명 옆에 MomBTI 유형 태그가 표시되어 유사 성향 엄마를 빠르게 식별
🔍
검색 & 페이지네이션
키워드 검색과 정렬로 원하는 정보를 빠르게 탐색, 쾌적한 피드 탐색 경험
Key Value
맘카페 의존 피로도를 줄이고, 유사 성향·상황의 엄마들과 연결해 깊은 정서적 공감대를 형성합니다.
🏠
Smart Home
LG 가전 연동
임신 주차와 컨디션에 맞춰 에어컨, 공기청정기 등 LG 가전이 최적의 물리적 환경을 자동으로 세팅합니다.
😴
꿀 무드 수면 루틴
에어컨 온도·풍량 자동 조절로 임산부에게 최적의 수면 환경 조성
👕
빨래 세탁-건조 연계
노터치 세탁-건조 자동 연계 루틴으로 임산부의 가사 부담 경감
🌬️
먼지 철벽 방어
공기청정기가 실내 공기질을 실시간 모니터링하고 자동 정화 루틴 실행
Key Value
엄마의 컨디션 변화에 맞춰 ThinQ 가전이 최적 실내 환경을 자동 조성합니다.

Personality
MomBTI 테스트

📋
성향 분석 테스트
간단한 문항으로 나만의 육아 성향을 파악하고 유형별 결과 제공
🏷️
커뮤니티 태그 연동
MomBTI 결과가 커뮤니티 작성자명 옆에 태그로 표시되어 유사 성향 엄마 식별
🤝
맞춤형 커뮤니티
유사 성향·상황의 엄마들과 연결되어 원활한 소통과 깊은 공감대 형성
Key Value
커뮤니티 탐색 효율화와 깊은 공감으로, 세상에 없던 라이프스타일 메이트를 만듭니다.
📝
Record
임신 다이어리
소중한 임신 기간의 매일을 사진과 글로 기록하고, 앨범으로 남기는 감성적 케어 기능입니다.
✍️
일기 작성·조회
날짜별로 임신 일기를 작성하고, 지난 기록을 언제든 돌아볼 수 있는 타임라인
📷
사진 첨부 기록
초음파 사진, 배 사진 등 소중한 순간을 이미지와 함께 기록
💝
감성 앨범
280일간의 기록이 하나의 앨범이 되어 출산 후에도 소중한 추억으로 간직
Key Value
기록을 통한 감성 경험을 극대화하고, 임신 여정을 특별한 추억으로 남깁니다.
💬

검증된 의사 정보

Key Value
검증된 의사 정보를 한곳에 모아 정보 탐색 피로도를 줄이고 신뢰할 수 있는 답을 제공합니다.
